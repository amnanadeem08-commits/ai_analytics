This is a normal static website on GitHub Pages — not Streamlit.

## Enable once in GitHub

1. Open https://github.com/amnanadeem08-commits/ai_analytics/settings/pages
2. **Build and deployment** → Source: **GitHub Actions**
3. Push to `main` or `master` — workflow deploys automatically

## Local preview

```bash
cd ai-services-web
npm install
npm run dev
```
