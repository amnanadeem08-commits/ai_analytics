## Edit the live site

Edit files in ai-services-web/src/, then build:

```powershell
cd ai-services-web
$env:VITE_BASE="/ai_analytics/"
npm run build
Copy-Item -Recurse dist\* ..\docs\ -Force
git add docs ai-services-web
git commit -m "Update website"
git push
```

Or push ai-services-web changes — GitHub Actions rebuilds docs/ automatically.
